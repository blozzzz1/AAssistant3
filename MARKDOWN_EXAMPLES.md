# Примеры Markdown форматирования

Теперь ваш AI помощник поддерживает полное Markdown форматирование! Вот что вы можете использовать:

## 1. Заголовки

```markdown
# Заголовок H1
## Заголовок H2
### Заголовок H3
#### Заголовок H4
##### Заголовок H5
###### Заголовок H6
```

## 2. Текстовое форматирование

```markdown
**Жирный текст**
*Курсив*
***Жирный курсив***
`встроенный код`
```

## 3. Списки

### Нумерованный список:
```markdown
1. Первый пункт
2. Второй пункт
3. Третий пункт
```

### Маркированный список:
```markdown
- Пункт A
- Пункт B
- Пункт C
```

### Вложенные списки:
```markdown
1. Основной пункт
   - Подпункт 1
   - Подпункт 2
2. Еще один основной пункт
   - Вложенный пункт
     - Еще глубже
```

## 4. Блоки кода

### Встроенный код:
```markdown
Используйте `console.log()` для вывода
```

### Блок кода с подсветкой синтаксиса:
````markdown
```python
def hello_world():
    print("Hello, World!")
```
````

## 5. Блок-цитаты

```markdown
> Это блок-цитата
> Она может быть многострочной
```

## 6. Горизонтальная линия

```markdown
---
***
___
```

## 7. Ссылки

```markdown
[Текст ссылки](https://example.com)
[Google](https://google.com)
```

## 8. Изображения

```markdown
![Alt текст](https://example.com/image.jpg)
```

## 9. Таблицы

```markdown
| Заголовок 1 | Заголовок 2 | Заголовок 3 |
|------------|-------------|-------------|
| Ячейка 1   | Ячейка 2    | Ячейка 3    |
| Ячейка 4   | Ячейка 5    | Ячейка 6    |
```

## 10. Комбинированный пример

```markdown
# Инструкция по использованию API

## Введение

API предоставляет доступ к следующим функциям:

1. **Аутентификация**
   - Получение токена
   - Обновление токена
2. **Работа с данными**
   - GET запросы
   - POST запросы

## Примеры кода

### Python
```python
import requests

response = requests.get('https://api.example.com/data')
print(response.json())
```

### JavaScript
```javascript
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data));
```

## Важные замечания

> **Примечание**: Всегда используйте HTTPS для защиты данных

---

## Контакты

Для получения дополнительной информации посетите [наш сайт](https://example.com)
```

## Тестирование

Чтобы протестировать поддержку Markdown, просто отправьте сообщение AI с любым из вышеперечисленных примеров, и он вернет отформатированный ответ!

### Поддерживаемые возможности:
✅ Заголовки (H1-H6)
✅ Жирный текст
✅ Курсив
✅ Встроенный код
✅ Блоки кода с подсветкой синтаксиса
✅ Нумерованные списки
✅ Маркированные списки
✅ Вложенные списки
✅ Блок-цитаты
✅ Горизонтальные линии
✅ Ссылки (открываются в новой вкладке)
✅ Изображения (можно кликнуть для увеличения)
✅ Таблицы
✅ Списки задач (чекбоксы)

### Стилизация:
- Темная тема с фиолетово-синими акцентами
- Подсветка синтаксиса для блоков кода
- Интерактивные элементы (ссылки, изображения)
- Красивое оформление таблиц
- Адаптивный дизайн
